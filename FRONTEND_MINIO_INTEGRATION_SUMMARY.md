# Резюме интеграции MinIO во Frontend

## Что было реализовано

### 1. Сервис для работы с изображениями (`src/services/ImageService.ts`)

- **uploadImage(file, folder)** - загрузка изображений в MinIO через API
- **deleteImage(imageUrl)** - удаление изображений из MinIO
- **validateImageFile(file)** - валидация файлов (тип, размер)
- **createImagePreview(file)** - создание предварительного просмотра
- **getFolderByContentType(contentType)** - автоматическое определение папки
- **handleUploadError(error)** - обработка ошибок

### 2. Универсальный компонент загрузки (`src/components/ImageUpload.tsx`)

- Предварительный просмотр изображений
- Валидация файлов на клиенте
- Прогресс загрузки
- Обработка ошибок с уведомлениями
- Просмотр и удаление текущих изображений
- Поддержка drag & drop
- Адаптивный дизайн

### 3. Интеграция в существующие страницы

#### Страница актеров (`src/pages/admin/ActorsPage.tsx`)
- Заменено поле ввода URL на компонент загрузки

- Автоматическое сохранение URL в модели

#### Новая страница спектаклей (`src/pages/admin/PerformancesPage.tsx`)
- Полноценная CRUD страница
- Загрузка постеров спектаклей
- Управление жанрами, продолжительностью, датами

#### Новая страница новостей (`src/pages/admin/NewsPage.tsx`)
- CRUD для управления новостями
- Загрузка изображений для новостей

### 4. Обновление роутинга (`src/App.tsx`)

- Добавлены новые маршруты для спектаклей и новостей

## Поддерживаемые типы контента

Система автоматически организует изображения по папкам:

- `actors` - фотографии актеров
- `perfomances` - постеры спектаклей
- `directors` - фотографии режиссеров
- `news` - изображения для новостей
- `archive` - архивные изображения
- `achievements` - изображения достижений

## Технические особенности

- Поддерживаемые форматы: JPEG, PNG, GIF, WebP

## Как использовать

### Базовое использование компонента

```typescript
import ImageUpload from '../components/ImageUpload';

<ImageUpload
    currentImageUrl={imageUrl}
    onImageUpload={handleImageUpload}
    onImageRemove={handleImageRemove}
    contentType="actors"
    maxSize={10}
    disabled={false}
/>
```

### Интеграция в форму

```typescript
const handleImageUpload = (imageUrl: string) => {
    setFormData(prev => ({
        ...prev,
        image_url: imageUrl
    }));
};

const handleImageRemove = () => {
    setFormData(prev => ({
        ...prev,
        image_url: ''
    }));
};
```

## Запуск

1. Убедитесь, что backend и MinIO запущены
2. Перейдите в папку frontend: `cd frontend/antrakt`
3. Установите зависимости: `npm install`
4. Запустите сервер: `npm start`
5. Откройте http://localhost:3000

## Доступные страницы

- **Главная**: http://localhost:3000/
- **Админ-панель**: http://localhost:3000/admin
- **Актеры**: http://localhost:3000/admin/actors
- **Спектакли**: http://localhost:3000/admin/performances
- **Новости**: http://localhost:3000/admin/news
